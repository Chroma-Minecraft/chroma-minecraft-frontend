## Apa itu Portable Gold Farm

Sebuah custom plugin dari Chroma Minecraft yang menghadirkan mekanisme baru yaitu Gold Farm yang memiliki mobilitas tinggi. Plugin ini menggantikan vanilla Gold Farm, sehingga vanilla Gold Farm dinonaktifkan pada server Chroma Minecraft untuk mengurangi kinerja server yang berlebihan pada vanilla Gold Farm.

## Spesifikasi

- Menggunakan sistem level dengan jumlah drop yang berbeda tiap levelnya.
- Tiap level dapat dibeli menggunakan obsidian. Harga tiap level adalah 11279 obsidian.
- Secara default, Portable Gold Farm akan berstatus Shared.
- Tiap pemain hanya bisa memiliki 1 Portable Gold Farm.
- Exp Booster dan Item Booster dapat dibeli melalui command **/donate**.

## Drop Table

Perhitungan drop item dan exp adalah sebagai berikut (pembulatan):

```
item = level * 0.625
exp = level * 5.1
harga = level * 11279

gold block  = ⌊ item / 81 ⌋
gold ingot  = ⌊ ⌊ item % 81 ⌋ / 9 ⌋
gold nugget = ⌊ ( item % 81 ) % 9 ⌉
```

Notes:

1. Perhitungan `gold block` akan drop apabila Portable Gold Farm memiliki level >= 130
2. Perhitungan `gold ingot` akan drop apabila Portable Gold Farm memiliki level >= 15
3. Perhitungan `gold nugget` akan drop apabila Portable Gold Farm memiliki level >= 1
4. Perhitungan `item` akan digunakan untuk nilai dasar drop
5. Perhitungan `exp` akan digunakan untuk drop experience
6. Perhitungan `harga` akan digunakan untuk harga jual gold farm pada level x
7. Level yang dimaksud adalah level Portable Gold Farm milikmu saat ini

Symbol:

1. ⌊x⌉ = Pembulatan nilai x (round)
2. ⌊x⌋ = Pembulatan nilai x ke bawah (floor)
3. ⌈x⌉ = Pembulatan nilai x ke atas (ceil)

## Booster

Booster memiliki kepada pemain untuk mendapatkan exp dan drop rate yang lebih tinggi pada Portable Gold Farm yang terhitung dalam pada perhitungan `item` pada rumus diatas. Booster dapat diakses melalui command **/donate**.
