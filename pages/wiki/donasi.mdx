Chroma Minecraft menjual cash dengan nama Chroma Cash (CC) yang digunakan sebagai
donasi untuk server ini. Player bebas menggunakan CC yang dimiliki untuk melakukan
pembelian pada fitur yang disediakan pada server Chroma Minecraft.

## Syarat & Ketentuan

1. Donasi pada layanan Chroma Minecraft bersifat sukarela artinya kami tidak memaksakan pemain untuk melakukan Donasi.
2. Donasi tidak dapat direfund karena sifat donasi itu sendiri sehingga pastikan apakah kalian benar-benar ingin melakukan Donasi telebih dahulu.
3. Pemain wajib membaca keterangan / deskripsi dari setiap item in game ( /donate ) sebelum melakukan pembelian menggunakan CC.
4. Apabila terjadi masalah saat pembelian item in game ( /donate ) bisa hubungi Admin via Discord.
5. Pemain bebas menggunakan CC untuk transaksi dengan pemain lain, resiko yang terjadi merupakan tanggung jawab pemain itu sendiri.

> Note: Syarat & Ketentuan berubah sewaktu-waktu, jadi pastikan untuk selalu membaca pesan ini sebelum melakukan donasi.

## Metode Pembayaran

**Harga 1 CC = Rp. 1.000**

Metode pembayaran:

1. \*Trakteer: https://trakteer.id/chromamc
2. GOPAY (Pay): 0812 3410 6350
3. Alfamart (Top-up GOPAY): 0812 3410 6350

> Note: Tambahan 10% lebih besar (contoh: jika ingin membeli 200 CC, maka perlu membayar Rp 220.000)

## Alur Pembayaran

> Diharapkan untuk join Discord [disini](https://discord.chroma-gaming.xyz) untuk mengirim bukti pembayaran

<br />

- Trakteer

```md
Pemain mengisi form pembelian (bisa dilihat contoh diatas)
Pemain membuka website https://trakteer.id/chromamc
Tekan tombol "kirimkan dukungan"
Isi jumlah CC yang diinginkan
Isi data nama dengan Username Minecraft kalian (contoh: Fakhri69)
Isi data pesan dukungan dengan "Pembayaran CC"
Pilih mode pembayaran
Centang opsi "tampilkan email saya pada kreator"
Pilih pembayaran sekali (Tip)
Lakukan proses pembayaran
Kirim form pembelian + bukti pembayaran pada #payment di Discord
```

<br />

- GOPAY (Pay)

```md
Pemain mengisi form pembelian (bisa dilihat contoh diatas)
Tekan tombol Pay pada aplikasi GOJEK
Masukkan nomor telepon 081234106350 (Atas nama Muhammad Fakhri Imaduddin)
Isi nominal yang dikirimkan
Isi data note dengan Username Minecraft kalian (contoh: Fakhri69)
Lakukan proses pembayaran
Kirim form pembelian + bukti pembayaran pada #payment di Discord
```

<br />

- Alfamart (Top-up GOPAY)

```md
Pemain mengisi form pembelian (bisa dilihat contoh diatas)
Lakukan proses Top-up GOPAY melalui kasir dengan nomor 081234106350 (Atas nama Muhammad Fakhri Imaduddin)
Kirim form pembelian + bukti pembayaran pada #payment di Discord
```
